Готовый комплекс серверного программного обеспечения LEMP (Linux Enginx MariaDB PHP)
====================================================================================
С помощью этой сборки на docker-compose можно разрабатывать веб-проекты на PHP и используя CMS, написанные на PHP

Данная сборка универсальна в плане переработок - можно поменять СУБД, версию PHP под требования. Это понадобится, если вам будет нужно локально работать над веб сайтом, который ранее находился на хосте с определёнными версиями пакетов

Ключевые возможности
--------------------
* Используется движок Nginx с базовой конфигурацией
* PHP 7.4.16 от Bitnami (с коробки собраны расширения, которые необходимы для комфортной работы)
* База данных — MariaDB 10.3
* PhpMyAdmin для комфортной работы с базами данных

Быстрый старт
-------------
Для начала нужно установить [Docker](https://www.docker.com/get-started)
Ещё помниться, что для Docker на Windows в настройках нужно указывать диск, где будут собираться контейнеры
Я не помню, где, т. к. пользуюсь Linux
1. Настраиваем в проекте файлы ./docker-compose, ./php/Dockerfile и ./nginx/site-name.conf по своему желанию. Последний файл так же можно переименовывать под имя вашего проекта
2. Переносим наш проект в каталог ./www/[site-name]
3. После настройки собираем командой `docker-compose build`
4. Далее запускаем `docker-compose up -d`
Опция \"-d\" нужна для запуска вне терминала или командной строки
5. Отслеживаем работу в логах и терминале. В Windows и Mac есть официальный GUI, в Linux я советую использовать [Protainer](https://documentation.portainer.io/v2.0/deploy/ceinstalldocker/)

Если ошибок нет, то всё готово. Приятного пользования!
